## 📂 Структура репозиторію (Виконані роботи)

* **[Лабораторна робота №1] Використання функцій** Реалізація математичних розрахунків за індивідуальним варіантом з використанням розгалужень (if/else) та вкладених циклів.
* **[Лабораторна робота №2] Робота зі вказівниками, рядками та файлами** Динамічне виділення пам'яті (`new`/`delete`), використання адресної арифметики замість класичних індексів, читання та запис даних у текстові файли (`ifstream`/`ofstream`).
* **[Лабораторна робота №3] Вказівники на функції та заголовні файли** Розбиття програми на окремі одиниці трансляції (заголовні файли `.h` та файли реалізації `.cpp`), використання механізму зворотного виклику (callback) для табулювання функцій.
* **[Лабораторна робота №4] Квадратичні алгоритми впорядкування масивів** Програмна реалізація базових алгоритмів сортування із квадратичною складністю $O(N^2)$: сортування вибором (Selection sort), вставкою (Insertion sort) та "бульбашкою" (Bubble sort).
* **[Лабораторна робота №5] Аналіз алгоритмів впорядкування** Реалізація шейкер-сортування (Shaker sort) та порівняльний аналіз часу виконання різних квадратичних алгоритмів на масивах великого розміру за допомогою функції `clock()`.
